# task-tracker# Python CLI Task Tracker

A lightweight, command-line based task management application written in Python. This was built as a mini-project to demonstrate file handling, loops, and basic CRUD (Create, Read, Update, Delete) operations.

## Features
* Add new tasks to your to-do list.
* View all current tasks.
* Mark tasks as complete.
* Persistent storage (tasks are saved to a `tasks.json` file automatically).

## How to Run
1. Ensure you have Python installed on your system.
2. Clone this repository.
3. Open your terminal or command prompt in the project folder.
4. Run the script using the following command:
   ```bash
   python main.py
